# 时钟待办 · Clock Todo Widget

这是一个完全由Claude Code + deepseek v4 pro做的小玩具，自己写着玩的。没有数据库，没有多设备同步。

一个简约的桌面时钟与待办事项悬浮窗，基于 Electron 构建，支持 Windows 系统。

## 功能

- **实时时钟** — 每秒刷新，显示当前时间和日期（含星期）
- **待办清单** — 添加、完成、删除待办事项，数据本地持久化
- **三款主题** — 深蓝 / 纯黑 / 纯白，适配不同桌面风格
- **窗口置顶** — 一键切换置顶/取消置顶，全屏工作时不会遮挡应用
- **系统托盘** — 最小化到托盘，托盘菜单支持显示/隐藏、切换主题、开机自启
- **毛玻璃效果** — 半透明背景 + 模糊效果，融入桌面
- **拖拽移动** — 按住时钟区域拖拽即可调整位置

## 技术栈

- **Electron** — 桌面应用框架
- **原生 HTML/CSS/JS** — 无框架、无打包工具，渲染进程零依赖

## 快速开始

```bash
# 安装依赖
npm install

# 终端启动
npx electron .
```

如需**静默启动**（不显示终端窗口），双击项目根目录下的 `start.vbs` 即可。

## 项目结构

```
├── start.vbs       # 静默启动脚本（双击运行，无终端窗口）
├── main.js         # Electron 主进程（窗口创建、托盘、IPC）
├── preload.js      # 预加载脚本（contextBridge）
├── index.html      # 渲染进程（UI + 样式 + 逻辑，全部单文件）
└── package.json
```
